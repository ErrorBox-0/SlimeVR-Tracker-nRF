# Tracker firmware for SlimeNRF

## Sensitivity Calibratrion 사용법
- 평면(책상 위 등)에서 각 축(xyz)마다 한바퀴 돌립니다.
- 360도 회전 후 나타나는 오차를 기록합니다.
- (ex) 56 -> 52 : 4
- 콘솔에서 다음과 같이 오차를 입력합니다.
- sens,4,5,-2
